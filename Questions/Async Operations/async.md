In JavaScript, operations fall into two categories:

| **Fast (Synchronous)** | **Slow (Asynchronous)**      |
| ---------------------- | ---------------------------- |
| Math calculations      | Network requests (API calls) |
| Variable assignments   | File reading                 |
| Loop iterations        | Timers (`setTimeout`)        |
| String manipulation    | User input (clicks, typing)  |
| Array methods          | Database queries             |
